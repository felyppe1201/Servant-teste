Perguntas Frequentes (FAQ)

Encontre as respostas para as dúvidas mais comuns sobre nossos serviços e produtos.

Pagamentos e Reembolsos

Quais métodos de pagamento são aceitos?

O Servant aceita cartões de crédito, débito e pagamentos via Pix.

Posso solicitar reembolso?

Sim, caso você não esteja satisfeito com o software, acesse nossa página de reembolso para mais detalhes.

Meu pagamento foi recusado. O que devo fazer?

Verifique se os dados do pagamento estão corretos ou entre em contato com o suporte do seu banco.

Instalação e Configuração

Como faço para instalar o Servant no meu dispositivo?

Basta seguir o guia passo a passo disponível na página de download.

Quais são os requisitos mínimos de sistema para rodar o Servant?

Os requisitos mínimos estão detalhados na seção de especificações técnicas no site.

O Servant é compatível com quais sistemas operacionais?

O software é compatível com Windows, macOS e Linux.

Funcionalidades e Operação

O que o Servant faz para proteger meu dispositivo?

Ele utiliza algoritmos avançados de detecção para proteger contra malwares, phishing e outras ameaças cibernéticas.

O software precisa de atualizações regulares?

Sim, para garantir que você tenha a proteção mais atualizada, recomendamos ativar as atualizações automáticas.

Posso personalizar as configurações de segurança?

Sim, o Servant oferece um painel de configurações intuitivo para você ajustar as opções de segurança conforme sua necessidade.
